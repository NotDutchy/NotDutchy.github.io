Project Boebot

During this project we worked in groups of 5-6 students and were tasked to create a self-driving robot. This was by interfacing with different sensors and using actuators to adjust the behavior of the robot. I also created a small GUI that communicates via a bluetooth module on the robot to send commands, aswell as a TV remote that send signals via infrared. This project built further upon the basics I learned in my first project. Here I used interfaces and started learning about design patterns. I used 3 layers to separate different parts of the software.

Sadly during this project we did not use git or Github yet. The source code is downloadable via the link below.
